 图书馆管理系统
基于 Java Web（JSP+Servlet+MySQL）开发的轻量级图书馆管理系统，适配Eclipse Mars、Tomcat 8.0.32、MySQL 5.5.14环境，支持图书管理、用户管理、借阅管理等核心功能。

 一、开发环境要求
本地部署前需准备好以下工具与环境，确保版本兼容以避免部署报错：
1.  JDK：1.8及以上版本（推荐jdk-8u461-windows-x64），作为Java程序的运行基础环境
2.  开发工具：Eclipse Mars版本（推荐eclipse-jee-mars-R-win32-x86_64），用于项目导入、代码编辑与调试
3.  Web服务器：Tomcat 8.0.32版本（推荐apache-tomcat-8.0.32），作为Web应用的部署与运行容器
4.  数据库：MySQL 5.5.14版本（推荐mysql-5.5.14-win32），用于存储系统的图书、用户、借阅等相关数据
5.  数据库驱动：mysql-connector-java-5.1.47.jar，适配MySQL 5.5.14版本的JDBC驱动，实现Java程序与数据库的连接
6.  可选数据库可视化工具：Navicat 163版本（推荐navicat163_mysql_cs_x64），方便可视化操作MySQL数据库，无需手动输入复杂SQL命令

 二、数据库初始化
1.  首先安装MySQL 5.5.14，运行对应的安装包完成部署，安装过程中设置并记录数据库用户名（默认通常为root）和登录密码，避免后续遗忘
2.  打开MySQL客户端（或可选的Navicat可视化工具），输入用户名和密码连接到MySQL服务
3.  找到项目根目录下sql文件夹中的library_db.sql脚本，将该脚本完整执行
4.  脚本执行完成后，会自动完成一系列操作：创建名为library_db的数据库、生成book图书表、user用户表、borrow_record借阅记录表三张核心数据表、插入管理员和普通用户的测试数据、插入示例图书数据、配置触发器实现时间字段的自动填充、为借阅记录表补充罚款金额和罚款状态字段，无需手动额外配置

 三、项目部署步骤
### 1. 导入项目到Eclipse Mars
1.  先解压Eclipse Mars的压缩包，双击解压目录中的eclipse.exe启动开发工具
2.  点击软件顶部菜单栏的File，选择下拉菜单中的Import选项
3.  在弹出的导入窗口中，依次展开General，选择Existing Projects into Workspace，点击Next
4.  点击Browse按钮，选中本地的图书馆管理系统项目根目录，勾选对应的项目名称
5.  最后点击Finish按钮，等待项目完成导入，导入成功后可在Eclipse的Package Explorer视图中看到项目完整结构

 2. 配置Tomcat 8.0.32
1.  先安装Tomcat 8.0.32，运行对应的安装包完成部署，记录下Tomcat的安装路径（例如C:\apache-tomcat-8.0.32）
2.  在Eclipse中点击顶部菜单栏的Window，选择下拉菜单中的Preferences选项
3.  在弹出的偏好设置窗口中，依次展开Server，选择Runtime Environments
4.  点击窗口右侧的Add按钮，在弹出的列表中展开Apache，选择Apache Tomcat v8.0，点击Next
5.  点击Browse按钮，选中之前记录的Tomcat 8.0.32安装路径，完成后点击Finish，再点击Apply and Close保存配置
6.  在Eclipse的Servers视图中，右键空白处，选择New，再选择Server
7.  在弹出的窗口中选择已配置的Tomcat 8.0，点击Next，勾选当前的图书馆管理系统项目，点击Add按钮将项目添加到服务器，最后点击Finish完成部署配置

3. 配置数据库连接
1.  在Eclipse的Package Explorer视图中，展开项目的src目录，找到com.util包下的DBUtil.java文件并打开
2.  在该文件中找到数据库连接相关的配置代码，将其中的用户名和密码修改为自己的MySQL账号和密码，URL地址保持默认即可（已适配library_db数据库）
3.  具体修改内容：确保URL为jdbc:mysql://localhost:3306/library_db?useSSL=false&characterEncoding=utf8，USER为自己的MySQL用户名（默认root），PWD为自己的MySQL登录密码123456
4.  检查项目的WEB-INF/lib目录下，是否已包含mysql-connector-java-5.1.47.jar数据库驱动包，若未包含，手动将该驱动包复制到该目录下，确保数据库连接正常

 五、核心功能
1.  图书列表展示：所有登录用户均可使用该功能，可查看系统中的全部图书信息，包括图书名称、ISBN编号、作者、出版社、出版日期、馆藏位置、库存数量以及图书状态等内容
2.  图书借阅：普通用户和管理员均可使用该功能，仅能借阅库存数量大于等于1的图书，借阅操作完成后，系统会自动扣减对应图书的库存数量，同时生成一条借阅记录，记录借阅用户、借阅图书、借阅日期和应还日期等信息
3.  图书归还：普通用户和管理员均可使用该功能，用户归还图书时，系统会自动计算该图书的逾期天数（若有逾期），并按照0.5元/天的标准计算逾期罚款金额，同时更新图书库存数量和借阅记录的实际还书日期、罚款金额、罚款状态等信息
4.  图书新增：仅管理员可使用该功能，管理员可通过该功能录入新的图书信息，包括ISBN编号、图书名称、作者、出版社、出版日期、馆藏位置和初始库存数量等内容，录入完成后图书信息会保存到数据库中，同时自动填充创建时间字段
